-------------------Mauritius Travel Guide Website-------------------
Welcome to the Mauritius Travel Guide Website! This project is designed to provide a comprehensive guide to Mauritius, highlighting its attractions, culture, and events. Our goal is to raise awareness among foreigners in Malaysia about the beauty and unique experiences Mauritius has to offer.

-----Getting Started-----
To get started with this project locally, follow these steps:
Run the homepage.html file in your preferred web browser to view the website.

-----Technologies Used-----
HTML: For the structure and content of the website.
CSS: For styling and layout.
JavaScript: For interactive elements and dynamic content.
